Interval arithmetic and exclusionTest

Integrate the interval arithmetic and the library with the exclusionTest.

LIB "intervalPureSingular.lib";

Define a ring

ring R=0,(x,y),lp;

Define an ideal $I$ with two ellipses as generators.

ideal I = 2x2-xy+2y2-2, 2x2-3xy+3y2-2;

This ideal is a zero-dimensional radical ideal what can be proved using the Singular functions vdim and radical from primdec.lib.

Now define the starting box. Therefore first define an interval i=[-3/2,3/2]

interval i=bounds(-3/2,3/2);i;

With this interval we can define a box:

$$ B=i \times i= [-3/2,3/2] \times [-3/2,3/2]$$

box B=list(i,i); B;

We take this box for as starting box for the algorithm exclusionTest. It can easily be seen that there are exactly four intersection points of the curves $2x^2-xy+2y^2-2$ and $2x^2-3xy+3y^2-2$. The algorithm states the same.

list result=exclusionTest(I,B,0); result;
